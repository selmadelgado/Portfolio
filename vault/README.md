# The Vault

A vault-style personal archive of artifacts from my career — credentials, letters, badges, mementos, and the stories behind them.

Each file in this vault catalogues a distinct era: the people, the places, the documents I kept. Together, they tell the story of how a kid in ninth grade journalism class became a designer and developer.

## Design

The vault uses a museum-archive aesthetic: warm cream backgrounds, brass accents, monospace metadata, italic editorial serif for headings, and subtle paper grain. Each artifact gets a "display case" with a placard caption written in the voice of an exhibit label. Built as static HTML/CSS — no frameworks, no build step.
